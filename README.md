## Setup

1. Clone this repo or upload it to Railway/GitHub.
2. Set environment variables:
   - `DISCORD_TOKEN` = Your Discord bot token
   - `CHANNEL_ID` = Your Discord channel ID
3. Run it:
   ```bash
   npm install
   node bot.js
   ```

## Endpoints

- `POST /roblox-to-discord`
- `GET /discord-to-roblox`
